Интернет-магазин навигаций в Германии

Период продвижения: август 2014 г. – ноябрь 2015 г.

Заказчик: Компания занимается интернет-продажей навигационных систем на автомобили премиум класса. Находится в России, продажа в Германии – посылками по почте.

Данные сайта до начала работ:

  • Разработка сайта – специально для немецкого сегмента мною был разработан новый сайт
  • Регион продвижения – Германия

Задачи продвижения:

  • Развитие сайта с нуля, включая разработку, заполнение контентом совместно с менеджером
  • Увеличение количества посетителей на сайт из поисковых систем;
  • Увеличение количества звонков с сайта

Проделанные работы:

  • Создание сайта, подключение модулей оплаты, личного кабинета, отзывов и т.д.
  • Внутренняя оптимизация сайта
  • Закупка ссылок в немецкоязычном сегменте

Результат:

После разработки сайта совместно с немецкоговорящим менеджером был составлен наиболее оптимальный список ключевых запросов (чуть больше 150) и проведена внутренняя оптимизация сайта. Также для более эффективной раскрутки был создан аккаунт по продаже товара, представленного на сайте, на Ebay.

Закупка ссылок на ресурс осложнялась высокой стоимостью ссылки на западном сегменте. В целях экономии были найдены близкие по тематике информационные ресурсы. После связи с администраторами данных ресурсов были написаны seo тексты для размещения на данных площадках. Также были найдены автомобильные форумы, развито общение на них для ненавязчивой рекламы продукции. Дополнительный трафик на сайт привлекался с помощью созданных корпоративных аккаунтов в Facebook и Twitter.

В итоге через год работы около 60% запросов было выведено в ТОП 10, а посещаемость сайта составила в среднем 130 посетителей в сутки.
